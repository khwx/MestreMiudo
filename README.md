# MestreMiúdo 🎮📚

**"Vamos aprender a brincar!"**

MestreMiúdo é uma plataforma educativa gamificada, desenhada para tornar a aprendizagem divertida e envolvente para crianças. Utilizando inteligência artificial e jogos interativos, a plataforma ajuda os "miúdos" a explorarem conhecimentos de forma lúdica.

## 🚀 Funcionalidades

-   **Jogos Interativos**:
    -   🔍 **Quiz**: Testes de conhecimento dinâmicos.
    -   🔠 **Jogo da Forca**: Expansão de vocabulário e ortografia.
    -   🃏 **Jogo da Memória**: Estimulação cognitiva e concentração.
    -   ⭕ **Jogo do Galo**: Raciocínio lógico e estratégia.
-   **IA Integrada**: Utiliza o Google Genkit para geração de conteúdos educativos adaptados.
-   **Voz e Som**: Integração com `edge-tts` para uma experiência auditiva imersiva.
-   **Dashboard Personalizado**: Acompanhamento do progresso do aluno.

## 🛠️ Tecnologias

A aplicação foi construída com tecnologias modernas de ponta:

-   **Framework**: [Next.js 15](https://nextjs.org/) (App Router & Turbopack)
-   **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
-   **Estilização**: [TailwindCSS](https://tailwindcss.com/) & [Radix UI](https://www.radix-ui.com/)
-   **Base de Dados**: [Supabase](https://supabase.com/) & [Firebase](https://firebase.google.com/)
-   **IA**: [Firebase Genkit](https://firebase.google.com/docs/genkit)
-   **Ícones**: [Lucide React](https://lucide.dev/)

## 📦 Instalação e Configuração

1.  **Clonar o repositório**:
    ```bash
    git clone https://github.com/khwx/MestreMiudo.git
    cd MestreMiudo
    ```

2.  **Instalar dependências**:
    ```bash
    npm install
    ```

3.  **Configurar variáveis de ambiente**:
    Crie um ficheiro `.env` baseado no `.env.example` e adicione as suas chaves do Supabase, Firebase e Google AI.

4.  **Iniciar o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```
    A aplicação estará disponível em `http://localhost:9002`.
