# **App Name**: MestreMiúdo

## Core Features:

- Personalized Learning Paths: Generates and tailors learning paths for each child based on their performance, focusing on areas of difficulty identified through their gameplay and question answering history. LLM acts as a tool by adaptively creating personalized quizzes and challenges.
- Interactive Game-Based Lessons: Engaging lessons for Portuguese, Mathematics, and Environmental Studies. The categories are tailored to 1st through 4th grade curriculums, including multiple choice questions, drag-and-drop exercises, and memory games.
- Progress Tracking and Gamification: Monitor children's progress, awarding points and badges that can be used to customize their avatar or unlock new games. Visual rewards that keep them motivated and encourage continuous learning.
- Audio Support for Questions: The text from the questions are available as audio, thus catering to various learning styles and accessibility needs of young children. Each question comes with a clear audio reading option, enabling better understanding, especially for those still learning to read.
- User Accounts and Avatars: Allow individual accounts for children, each featuring a customizable avatar. Accounts track progress and current level in each subject area. Personalization helps make the experience unique to each user.

## Style Guidelines:

- Primary color: Cheerful blue (#64B5F6) to convey trust and a positive learning atmosphere.
- Background color: Soft, desaturated blue (#E3F2FD) to minimize distractions.
- Accent color: Energetic yellow (#FFEB3B) for interactive elements and highlights, creating visual interest.
- Font pairing: 'Playfair' (serif) for headlines paired with 'PT Sans' (sans-serif) for body to make the text appealing.
- Use playful and intuitive icons that align with the educational content of each subject area. Icons should be colourful, simple, and easily recognizable for young children.
- Subtle animations for feedback and transitions that aren't too distracting, using animated rewards to celebrate progress. Micro-interactions to confirm answers.
