# Instruções para o Assistente

## Regra Importante
- **Sempre** commitar e enviar (push) quaisquer alterações para o GitHub após completar uma tarefa
- Antes de enviar, executar testes, lint e typecheck
- Usar mensagens de commit descritivas em português

## Comandos do Projeto
- Lint: `npm run lint`
- Typecheck: `npm run typecheck`
- Testes: `npm test` (se existirem)
